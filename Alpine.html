Présentation du projet

Ce premier projet etait l'un des projet individuel au choix à réaliser dans le cadre de l'epreuve Technique E6 du BTS SIO.

Ce projet avait pour but de se metre dans la developpeur et concevoir une plateforme de réservation dédiée aux activités de montagne.

Aventures Alpines permet aux utilisateurs de découvrir et réserver leurs aventures en montagne.

Période : De Janvier à Mars 2026
Lieu : Domicile ou Pôle BTS

Missions réalisées

  • Conception de l'application
  • Documentation de l'application
  • Maintenance et amélioration de fonctionnalités existantes
  • Tests et corrections
  • Rédaction de documentation technique

Technologies et outils utilisés

React (Frontend)
Node.js (Backend)
MySQL (Base de données)
GitHub (Versioning)
JavaScript (Langage)
API Rest (Architecture)
